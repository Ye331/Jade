# Shanhai Gate Platform Components

These prefabs are ready-to-place modular platform pieces for the cave/double-jump room.

- `SG_Platform_05x065.prefab` uses `SG_Fit_Platform_5x065.png`
- `SG_Platform_10x1.prefab` uses `SG_Fit_Platform_10x1.png`
- `SG_Platform_20x1.prefab` uses `SG_Fit_Platform_20x1.png`
- `SG_Platform_40x1.prefab` uses `SG_Fit_Platform_40x1.png`

Each prefab includes a `SpriteRenderer` and a `BoxCollider2D`. Collider height is intentionally focused on the stone top/body so vines and overhang details do not become accidental collision.
